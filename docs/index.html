MagicMove

Move folders to another drive and leave behind a symbolic link. Applications keep working — they never know the folder moved.

📁

Move Anywhere

Relocate folders to any drive or directory. The original path gets a symbolic link so everything keeps working.

↺

One-Click Revert

Changed your mind? Revert a move instantly — the symlink is removed and the folder returns to its original location.

🔒

Admin Enforced

Creating symbolic links requires administrator privileges. MagicMove requests elevation via UAC automatically.

💾

Persistent Tracking

All moved folders are remembered across sessions. Settings are stored in %APPDATA% so they survive reinstalls.

●

Health Status

Each entry shows a live status indicator — green when the symlink is healthy, red when something needs attention.

⚡

Lightweight

A single-purpose tool with no bloat. Built with WinForms on .NET — fast startup, tiny footprint.

How It Works

1

Pick a Folder

Select the folder you want to relocate and choose a destination.

2

Folder Moves

The folder is moved to the new location on your target drive.

3

Symlink Created

A symbolic link is placed at the original path pointing to the new location. Apps see no difference.

Requirements

Windows 10 or later

.NET 9.0 Runtime (included in self-contained builds)
